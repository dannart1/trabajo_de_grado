  ACERCA DE NOSOTROS

Este trabajo de grado lleva el nombre de "Prototipo de servicio web para el seguimiento y control del cultivo de ají cayenne mediante una arquitectura IoT" y tiene como objetivos específicos los siguientes:

  • Realizar el estado del arte de los servicios y/o aplicaciones web para el control y monitoreo de cultivo mediante arquitecturas IoT en el sector de la agricultura.
  • Construir la arquitectura de software basada en microservicios para el control y monitoreo en el cultivo de ají cayenne.
  • Construir el prototipo de servicio web sobre una arquitectura IoT para el monitoreo de humedad, control de plagas y conteo de plantas en el cultivo de ají cayenne.
  • Validar el prototipo de servicio web mediante un escenario de pruebas.

    • Estos objetivos fueron planteados con el fin de responder a la pregunta planteada ¿Cómo diseñar un servicio web que permita determinar la humedad del suelo, enfermedades existentes y realizar conteo de plantas ají cayenne en el terreno ubicado en Tres Esquinas de Tuluá, Valle?
